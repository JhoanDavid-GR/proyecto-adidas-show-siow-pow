# Proyecto Adidas - Share of Wallet

Autor: Jhoan David Fernando Gutierrez Rubiano, Juan Garcia, Sergio Aguilar

Maestría en Ciencias de Datos

## Descripción

Este repositorio contiene los scripts y resultados utilizados para el desarrollo de la tesis de maestría orientada a la estimación de Share of Wallet (SoW), Size of Wallet (SioW) y Potential of Wallet (PoW) para clientes Adidas utilizando datos transaccionales de Amazon.

## Archivos

- code1_adidas.Rmd: preparación inicial de datos.
- code2_adidas.Rmd: construcción de matrices y variables de modelado.
- script3_analisis_descriptivo_adidas_5.Rmd: análisis descriptivo y exploratorio.
- Script4 - Estimación del modelo.Rmd: estimación y comparación de modelos Poisson, Binomial Negativa, ZIP y POGIT.

## Orden de ejecución

1. code1_adidas.Rmd
2. code2_adidas.Rmd
3. script3_analisis_descriptivo_adidas_5.Rmd
4. Script4 - Estimación del modelo.Rmd
